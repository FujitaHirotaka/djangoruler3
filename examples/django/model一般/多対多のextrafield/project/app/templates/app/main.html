ManyToManyField

[参考ページ]Django公式document

AとBの所属関係を記述するだけならManytoManyFieldでもできる。

AがBにいつから、どれくらいの間所属したかなど、AとBの詳細な関係性までを記述するためにはmanytomanyfieldだけでは記述できない。

（manytomanyfieldだけで記述しようとすると、できないことがわかると思う）

このような関係性を追加で記述するのがmanytomanyfieldにおけるextrafield

基礎的な例(_set.allの使い方）

直接関係性を記述していないが、きちんとお互い多対多の関係になっている

{% for i in player_list %} {{i}}
{{ i.team_set.all }}
{% endfor %}

基礎的な例(allの使い方）

{% for i in team_list %} {{i}}
{{ i.players.all }}
{% endfor %}

基礎的な例3

{% for i in membership_list %} {{i}}
{{ i.player.name }}
{{ i.team.name }}
{{ i.relationship }}

{% endfor %}

path converterの使い方

パスコンバーターの値：{{ value }}

このように汎用viewには自動的にパスコンバーターのパラメーターは渡されるようだ。

http://127.0.0.1:8080/app/index/123にアクセスするとパスコンバーターのintの作用を確認することができる

http://127.0.0.1:8080/app/index/あああいいいにアクセスするとパスコンバーターのstrの作用を確認することができる

http://127.0.0.1:8080/app/index/aaa-bbb_123にアクセスするとパスコンバーターのslugの作用を確認することができる

http://127.0.0.1:8080/app/index/ok/goodにアクセスするとパスコンバーターのpathの作用を確認することができる

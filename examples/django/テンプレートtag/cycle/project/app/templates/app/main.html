cycle

[参考ページ]Django公式document

(概要）{% verbatim %}{% cycle 変数１　変数2　変数3 %}{% endverbatim %}で順に一つずつ変数の値を出力します

for文との組み合わせで使う

１．基本的な使用例("aaa","bbb",などが変数)

{% for i in list %} {% endfor %}
{{i}}. {% cycle "aaa" "bbb" "ccc" "ddd" "eee" %}

２．基本的な使用例（変数でもＯＫ）

{% for i in list %} {% endfor %}
{{i}}. {% cycle variable0 variable1 variable2 variable3 variable4 %}

３．autoescapeをoffにしたい場合

{% for i in list %} {% endfor %}
{{i}}. {% autoescape off %}{% cycle variable0 variable1 variable2 variable3 variable4 %}{% endautoescape %}

４．変数と文字列を混ぜて使用

{% for i in list %} {% endfor %}
{{i}}. {% autoescape off %}{% cycle variable0 variable1 "bbbb" variable2 "aaaa" %}{% endautoescape %}

５．変数や文字列を再利用したい場合

{% for i in list %} {% endfor %}
{{i}}. {% autoescape off %}{% cycle variable0 variable1 "bbbb" variable2 "aaaa" as revariable %}{% endautoescape %} {{ revariable }}

6．cycleを宣言したいだけでその場では値を生成したくない場合（silentの使用)

{% for i in list %} {% cycle variable0 variable1 "bbbb" variable2 "aaaa" as revariable silent%} {% endfor %}
{{i}}. {% autoescape off %}{{ revariable }}{% endautoescape %} {{ revariable }}

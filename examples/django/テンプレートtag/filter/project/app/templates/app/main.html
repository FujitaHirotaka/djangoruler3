filter

[参考ページ]Django公式document

タグブロック内のコンテンツを、1 つまたは複数のフィルタに通します。

基本例１(linebreaksのみ）

{% filter linebreaks %} FIRST LINE SECOND LINE {% endfilter %}

基本例２(linebreaksとlower）

{% filter linebreaks|lower %} FIRST LINE SECOND LINE {% endfilter %}

基本例３(lowerのみ）

{% filter lower %} FIRST LINE SECOND LINE {% endfilter %}

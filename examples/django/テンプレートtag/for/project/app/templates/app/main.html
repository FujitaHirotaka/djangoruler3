基本例0(リストへのアクセス)

リストへのアクセスはlist[0][1]のような形でなく、list.0.1のような形でアクセスする

{{ list.0 }}
{{ list.1.2 }}
{{ list.0.1 }}
{{ list.0.2 }}
{{ list.0.3 }}

基本例1(リストの展開)

{% for i in list %} {{ i }}
{% endfor %}

基本例2(reversed)

{% for i in list reversed %} {{ i }}
{% endfor %}

基本例3(入れ子、リストの中のリストにアクセスする）

ネストされたリストの展開も可能

{% for i in list %} {% for ii in i %} {{ ii }}
{% endfor %} {% endfor %}

基本例4(リストの中のリストにアクセスするその２)

ネストされたリストについて for x,y,z in listのような形で展開できる

{% for x,y,z in list %}

This is a point at ({{ x }}, {{ y }}, {{ z }})

{% endfor %}

基本例５(辞書の展開)

for i in jishoのような形で展開した場合、キーが展開される

{% for i in jisho %}

{{ i }}

{% endfor %}

基本例6(辞書の展開その２)

for i in jisho.keysのような形で展開した場合、キーが展開される

{% for i in jisho.keys %}

{{ i }}

{% endfor %}

基本例７(辞書の展開その３)

for i in jisho.valuesのような形で展開した場合、バリューが展開される

{% for i in jisho.values %}

{{ i }}

{% endfor %}

基本例8(辞書の展開その4)

for i, ii in jisho.itemsのような形で展開した場合、キーとバリューが展開される

{% for i, ii in jisho.items %}

{{ i }}:{{ ii }}

{% endfor %}

基本例9(辞書へのアクセス)

jisho["key"]のような形でアクセスするのではなく、jisho.keyのような形でアクセスする

辞書やリストが入れ子になった複雑なものでも基本的にこのようなやり方ですべてアクセスできるようだ。

{{ jisho.bbb}}

{{ jisho2.2}}

{{ jisholist.1.ddd}}

firstof

[参考ページ]Django公式document

(概要）{% verbatim %}{% firstof 変数１　変数２　変数３　・・・・%}{% endverbatim %}で後ろにある変数のうち、Trueである最初のものを出力

基本例1

{% verbatim %}{% firstof False True %}{% endverbatim %}
{% firstof False True %}

基本例2

{% verbatim %}{% firstof "" 0 0.0 1.0 %}{% endverbatim %}
{% firstof "" 0 0.0 1.0 %}

基本例3

{% verbatim %}{% firstof karamojiretu zero zerofloat karalist "OK" %}{% endverbatim %}
{% firstof karamojiretu zero zerofloat karalist "OK" %}

基本例4

{% verbatim %}{% firstof karamojiretu zero zerofloat karalist "OK"|safe %}{% endverbatim %}
{% firstof karamojiretu zero zerofloat karalist "OK"|safe %}

autoescape tag

[参考ページ]Django公式document

(注意）autoescapeはデフォルトでオン

1.autoescapeの設定なし（すなわちデフォルト）

{{ variable }}

2.autoescape off

{% autoescape off %} {{ variable }} {% endautoescape %}

3.autoescape on(すなわちデフォルトとかわらない）

{% autoescape on %} {{ variable }} {% endautoescape %}

4.テンプレートフィルタのsafeを使ってautoescapeをoffすることも可能

{{ variable|safe}}
